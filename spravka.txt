Справка.

Данная программа предназначена для вычисления квадратных корней из арифметичеких или комплексных чисел.

Выберите вкладку Арифметические или Комплексные кликнув по ней правой кнопкой мыши.

Во вкладке Арифметические:

Введите число, из которого вам требуется извлечь корень при помощи кнопок с цифрами.
Если число не целое - используйте Точку(кнопку с символом ".").
Кнопка с символом "CE" стирает все введенное или полученное значение.
Кнопка с символом "С" стирает один сивол с права налево из введенного значения или стирает выведенное значение полностью.
Нажатие кнопки Равно (с символом "=") выводит результат.

Во вкладке Комплексные:

Комплексные числа вводятся в формате ("целая часть")+ ("коэффициент при i")i
Пример: 4+2i
Выберите при помощи кнопок "Вводить действительную часть" и "Вводить мнимую часть" окно для ввода соответсвующих числовых значений.
Введите число, из которого вам требуется извлечь корень при помощи кнопок с цифрами.
Если число не целое - используйте Точку(кнопку с символом ".").
Если число отрицательное - используйте Минус (кнопку с символом "-").
Кнопка с символом "CE" стирает все введенное или полученное значение.
Кнопка с символом "С" стирает один сивол с права налево из введенного в выбранное поле значения или стирает выведенное значение полностью.
Нажатие кнопки Равно (с символом "=") выводит результат.

При помощи Меню можно:
Выбрать язык приложения: Меню -> Язык -> English или Русский
Посмотреть информацию о приложении:Меню -> О программе
Связаться с поддержкой: Меню -> Помощь -> Форум поддержки
Посмотреть справку: Меню -> Помощь -> Справка
Выйти из приложения: Меню -> Выход
